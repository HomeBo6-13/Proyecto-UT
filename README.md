# Proyecto-UT
Desarrollar un sistema de inventario para una universidad enfocado en la gestión y control de activos físicos dentro de sus instalaciones. El sistema debe permitir registrar y administrar todos los objetos disponibles en los edificios, tales como sillas, escritorios, computadoras, televisores, entre otros.

El objetivo principal es mantener un control preciso del inventario para prevenir la pérdida de materiales y mejorar la organización institucional. Cada objeto debe estar registrado con información relevante y asociado a un identificador único.

El sistema también debe contemplar la gestión de préstamos de los recursos. Para ello, habrá profesores responsables asignados a ciertas aulas, quienes se encargarán de autorizar y registrar los préstamos de los objetos. Cada préstamo debe generar un registro que incluya los datos de la persona que solicita el objeto, el artículo prestado, la fecha, y demás información necesaria para el seguimiento.

Adicionalmente, todos los objetos registrados deberán contar con tecnología NFC, lo que permitirá rastrear y validar cada elemento mediante un identificador único, asegurando que coincida con la información almacenada en el sistema.

El sistema debe ser capaz de integrar el control de inventario, gestión de préstamos y rastreo mediante NFC en una sola solución eficiente y fácil de utilizar.
